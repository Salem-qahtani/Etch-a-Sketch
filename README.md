# Etch-a-Sketch

A browser-based drawing pad built with vanilla JavaScript, HTML, and CSS. Part of [The Odin Project](https://www.theodinproject.com/) Foundations curriculum.

## Features

- **16×16 default grid** generated dynamically with JavaScript
- **Color mode** — pick any color and draw with it
- **Random mode** — each square gets a random RGB color on hover
- **Eraser** — erase specific squares back to white
- **Clear button** — resets the entire grid
- **Custom resolution** — enter any size from 1 to 100 and regenerate the grid

## How to Use

1. Select a drawing mode (Color, Random, or Eraser)
2. Click and drag across the grid to draw
3. Use the color picker to choose your color in Color mode
4. Enter a number and click **Generate** to resize the grid
5. Click **Clear** to wipe the canvas

## Built With

- HTML
- CSS (Grid layout)
- Vanilla JavaScript (DOM manipulation, event listeners)
