# Etch-a-Sketch
 
A browser-based drawing pad built with vanilla JavaScript, HTML, and CSS. Part of [The Odin Project](https://www.theodinproject.com/) Foundations curriculum.
 
## Live Preview
 
[View on GitHub Pages](https://salem-qahtani.github.io/Etch-a-Sketch)
 
## About
 
A drawing pad inspired by the classic Etch-a-Sketch toy. Users can draw on a customizable grid using different modes including a color picker, random colors, and an eraser. Built to practice DOM manipulation and event handling with vanilla JavaScript.
 
## Features
 
- **Color Mode** — Pick any color and draw with it
- **Random Mode** — Each square gets a random RGB color on hover
- **Eraser** — Erase specific squares back to white
- **Clear** — Resets the entire grid
- **Custom Resolution** — Enter any size from 1 to 100 and regenerate the grid
 
## Built With
 
- HTML
- CSS (Grid layout)
- Vanilla JavaScript (DOM manipulation, event listeners)
 
## What I Learned
 
- Generating and manipulating DOM elements dynamically with JavaScript
- Handling mouse events (mousedown, mouseover, mouseup) for click-and-drag drawing
- Using CSS Grid with dynamic `grid-template-columns` for flexible grid sizing
- Managing UI state across multiple drawing modes
